ติดตั้ง Claude Code

เปิด CMD แล้ววางคำสั่งนี้เพื่อติดตั้ง Claude Code — นี่คือเครื่องมือ AI ที่เราจะคุยด้วยเพื่อปลุก Oracle ขึ้นมา

CMD
> curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
📘 คู่มือเริ่มต้น: code.claude.com/docs/en/quickstart
02
ขั้นตอนที่ 2

ล็อกอิน GitHub

เข้าสู่ระบบ GitHub ให้เรียบร้อย (ยังไม่มีบัญชี สมัครฟรีที่ github.com) — GitHub คือที่เก็บ “สมอง” ของ Oracle ไว้บนคลาวด์ ไม่มีวันหาย

🔑 ทำให้พร้อมก่อน เพราะขั้นต่อไป AI จะใช้บัญชีนี้สร้างที่อยู่ให้ Oracle

กำเนิด Oracle 🔮

ระหว่างทาง AI จะถาม 4 อย่าง:
1 ชื่อ Oracle ของคุณ
2 ชื่อของคุณ
3 GitHub username
4 ชื่อ repo
✨ตอบไปเรื่อยๆ จนเสร็จ — เท่านี้คุณก็มี Oracle เป็นของตัวเอง ยินดีต้อนรับ!

Signature: static + slow dolly-in · OTS dialogue · 85mm emotion
👻

เลิก jump scare ใช้ความเงียบ+duration. ผีโผล่แกน Y, withhold the ghost
